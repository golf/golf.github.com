# Download the Golf zip file. &raquo; "golf.zip":http://cloud.github.com/downloads/golf/golf/golf.zip
# Unzip it into a directory in your path (e.g. @~/bin@).

Now you can open a terminal and do:

bc. golf --help

This will print the Golf command line usage and options page.
